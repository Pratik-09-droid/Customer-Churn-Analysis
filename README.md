# 📊 Customer Churn Analysis

## Project Overview

This project analyzes customer churn in a telecom company using Python. The objective is to clean the data, perform exploratory data analysis (EDA), identify the factors affecting customer churn, and build machine learning models to predict customer churn.

- 🔄 Task 2: Exploratory Data Analysis
